## 2. 확정된 결정사항 (되돌리지 말 것)

### 편집 잠금 기능
- **동작 방식**: 상세 진입 시 `lockTicket` 호출. 이미 잠겨 있으면 **차단 팝업(확인 버튼만) → 목록으로** (읽기전용 모드 아님). 저장/취소/삭제/이탈 시 잠금 해제.
- GAS: `COL.LOCKED_AT=16`(Q열), `lockTicket`/`unlockTicket`/`checkLock`, 30분 만료.
- 목록 자물쇠: 잠긴 항목 티켓번호 앞에 🔒 (첨부 컬럼 뒤). `locked_at` 30분 이내일 때 표시.
- **잠긴 항목 클릭 시**: GAS 재조회 없이 **캐시(locked_at)로 즉시 팝업** 띄우고 상세 진입 안 함. 상세의 lockTicket은 "아이콘 뜨기 전 방금 잠긴" 경우만 잡는 최종 안전장치.
- **자동 자물쇠 폴링**: 목록에서 20초마다(`LOCK_POLL_MS`) 잠금 상태 조회 → **아이콘만** 추가/제거(전체 재렌더 X). 비활성 탭은 스킵, 복귀 시 즉시 1회.
- **내가 방금 푼 항목 억제**: 저장/취소 후 목록 돌아올 때 sessionStorage `dqa_released_row` 힌트로 자기 자물쇠를 서버 반영 전까지 숨김 (`suppressLockRowId`, `isLockedForDisplay`).
- 잠금 해제는 `sendBeacon`(비차단) — 이동 즉시. `beforeunload`가 백업.

### 드래그앤드롭 실시순서
- 맨 오른쪽 핸들(⠿) 전용 컬럼. **핸들에 mousedown 했을 때만 그 행 `draggable=true`** (tr 고정 draggable 금지 — dragstart의 e.target이 tr라 핸들 판별 불가했던 버그).
- 드롭 위치 **녹색 인디케이터 줄**(`.drop-above`/`.drop-below`, td box-shadow `#15803d`).
- 실시순서 **드롭다운도 유지**(드래그와 병행). WW/MVN 그룹 내에서만.
- **그룹 이동 시(WW↔MVN 포함) priority 초기화** — 중복 방지.

### UI/UX
- 기본 탭: localStorage 없으면 **최신 버전(sort_order 최대)** 자동 선택.
- COL_WIDTHS: `['24px','110px','','110px','70px','110px','120px','70px','80px','44px']`, 테이블 min-width 940px.
- 담당자/진행상태 셀렉트 간격: status-cell padding-left 14px, 컬럼 넓힘으로 확보.
- 인라인 셀렉트 모양 = 등록폼 셀렉트와 통일(사각 6px + 테두리), 색상/효과 유지.
- 진행중 행: 연노랑 배경 + 앰버 보더, hover 시 진한 노랑. status-select 진행중=파랑 배경.
- **모든 행 hover = 행 전체 강조**(`#eff6ff`), 셀 개별 강조 제거(이슈명/확인버전은 클릭 커서만).
- **커스텀 툴팁**(`.app-tooltip`, body 부착): 클립(첨부 대표 파일명)·자물쇠(편집중)·이슈명. **요소 위쪽** 표시(커서가 글씨 안 가림), native title 풍(옅은 노랑+테두리). `data-tip` 속성 사용. `setupTooltips()`.
- 자물쇠 아이콘: 절대위치 + 티켓번호칸 좌측 여백 26px 고정으로 행 정렬 유지. cursor:default(이모지 I-beam 방지).
